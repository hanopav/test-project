## Quick Start

Get up and running in minutes:

```bash
# 1. Clone the repo
git clone https://github.com/YOUR-USERNAME/YOUR-REPO.git

# 2. Install dependencies
cd YOUR-REPO
npm install

# 3. Run it
npm start
```

> **Tip:** You'll need Node.js 18+ installed. Check with `node -v`.
